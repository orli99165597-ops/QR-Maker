# AI Monetization Daily Research

이 폴더는 AI 관련 수익화 방법을 매일 조사하고 보고서로 저장하기 위한 작업 공간입니다.

## 실행 기준

- 매일 한국시간 오전 12시에 실행
- 하루에 중복되지 않는 AI 수익화 방법 2가지를 조사
- 기존에 기록된 방법과 겹치지 않도록 `methods_seen.md`를 먼저 확인
- 결과 보고서는 `reports/YYYY-MM-DD.md` 형식으로 저장

## 보고서 작성 원칙

각 방법마다 아래 항목을 포함합니다.

- 방법 이름
- 핵심 아이디어
- 수익 모델
- 필요한 준비물
- 실행 난이도
- 예상 첫 실험 단계
- 참고 출처 링크

새로 조사한 방법은 보고서 작성 후 `methods_seen.md`에도 추가합니다.
